# Quiz Management System (Go)

A command-line Quiz Management System built using Go.

## Features
- Admin can create quizzes
- Users can attempt quizzes
- Questions stored in JSON
- Simple CLI interface

## Technologies Used
- Go (Golang)
- JSON file storage

## How to Run

1. Clone the repository
git clone https://github.com/smeranil11-afk/Quiz-Management-System.git

2. Navigate to the folder
cd Quiz-Management-System

3. Run the program
go run main.go
